eventenrollment Enrolment Moodle Plugin
==============================

Features
--------

This enrolment plugin lets you automatically enrol users in the desired courses when they sign up to Moodle.

For example, you can set it to enrol all your new users to the "Moodle intro" course.

Installation / Configuration
----------------------------

1. Add the plugin into /enrol folder.
2. Enable the enrolment plugin and set it up. It is quite similar to other enrolment plugins.

Use
----
Enable the plugin in the courses you want to auto-enrol users on sign up.
Once a new user is added to Moodle, he will be automatically enroled in the courses that have the eventenrollment plugin enabled.
